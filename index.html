Glosario de Desarrollo Web

# Término Definición
1HTMLLenguaje de marcado para estructurar contenido web.
2CSSLenguaje de estilos para definir la apariencia de un sitio.
3JavaScriptLenguaje de programación que añade interactividad.
4PHPLenguaje del lado del servidor para crear contenido dinámico.
5MySQLGestor de bases de datos relacionales muy utilizado.
6DOMModelo de objetos del documento HTML.
7APIConjunto de funciones que permite comunicación entre sistemas.
8AJAXTécnica para actualizar contenido web sin recargar la página.
9JSONFormato de intercambio de datos ligero y legible.
10XMLLenguaje de marcado para estructurar información.
11POSTMétodo HTTP para enviar datos al servidor.
12GETMétodo HTTP para obtener datos del servidor.
13SESSIONMecanismo para almacenar información temporal de usuarios.
14COOKIEArchivo pequeño para guardar datos del usuario en su navegador.
15isset()Función en PHP para verificar si una variable está definida.
16echoInstrucción en PHP para mostrar texto o variables.
17varDeclaración de variable en JavaScript (obsoleta).
18letDeclara variables con alcance de bloque en JS.
19constDeclara constantes en JavaScript.
20functionDefine una función en JavaScript o PHP.
21arrayEstructura para almacenar múltiples valores.
22if/elseEstructura de control para condiciones lógicas.
23forBucle que repite código un número determinado de veces.
24whileBucle que se ejecuta mientras se cumpla una condición.
25formElemento HTML para capturar información del usuario.
26inputElemento HTML para ingresar datos.
27buttonBotón que puede enviar formularios o ejecutar código.
28selectDesplegable para seleccionar una opción.
29labelEtiqueta asociada a un campo de formulario.
30spanContenedor en línea sin estilo por defecto.
31divContenedor en bloque para agrupar elementos.
32classAtributo HTML para aplicar estilos CSS.
33idIdentificador único para un elemento HTML.
34hrefAtributo que define un enlace en HTML.
35srcFuente de una imagen, video o script.
36altTexto alternativo para imágenes.
37metaDefine metadatos como charset o viewport.
38SEOOptimización para motores de búsqueda.
39URLDirección de un recurso en Internet.
40IPDirección numérica de un dispositivo en la red.
41DominioNombre amigable que apunta a una dirección IP.
42DNSTraduce dominios a direcciones IP.
43ServidorComputadora que aloja servicios o sitios web.
44HostProveedor que almacena tu sitio en Internet.
45SSLProtocolo que cifra las comunicaciones en una web.
46HTTPSVersión segura del protocolo HTTP.
47FrontendParte visible y con la que interactúa el usuario.
48BackendParte lógica del sistema, en el servidor.
49FrameworkConjunto de herramientas para facilitar el desarrollo.
50CMSSistema de gestión de contenidos como WordPress.
